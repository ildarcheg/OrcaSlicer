# orca-cli Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `orca-cli.exe`, a 3MF composer CLI that clones a known-good reference 3mf and applies plate, object, filament, and per-object config mutations, producing files that open cleanly in the OrcaSlicer GUI.

**Architecture:** Thin orchestrator over libslic3r. Clone-and-mutate only; no synthesize-from-scratch, no preset overrides. Every save is followed by a runtime invariant guard (zip relationships, plate thumbnails, vector-config roundtrip) and e2e tests additionally assert archive-level XML invariants. Implementation is decomposed into 8 phases (P0–P7) each handled by one subagent and gated by a manual GUI smoke test before the next fires.

**Tech Stack:**
- C++17, CMake (Visual Studio 2022 / v143 on Windows).
- libslic3r (`store_bbs_3mf`, `load_bbs_3mf`, `Model`, `DynamicPrintConfig`, `PlateData`).
- Vendored CLI11 (header-only) for arg parsing.
- Catch2 v2 (already in repo) for unit/round-trip/e2e tests.
- `boost::process` (already linked by libslic3r deps) for e2e subprocess spawn.
- Minimal in-tree PNG writer (256 bytes of solid-color RGBA, deflate via zlib already in the dep tree).

**Source of truth for the contract:** `docs/superpowers/specs/2026-05-19-orca-cli-design.md`. Where this plan says "per spec § N", read that section before implementing.

## Cross-cutting conventions

- **Test framework:** Catch2 v2. Tests use `TEST_CASE("orca-cli: <desc>", "[orca-cli][<phase>]")` with the tag bracket. ASCII `->` not `→` in names (G8). Never `Approx`; use `WithinAbs` / `WithinRel` for floats (per `tests/CLAUDE.md`).
- **Skip on missing fixture (Catch2 v2 has no `SKIP`):** `if (!fs::exists(REF_3MF)) { SUCCEED("Skipped: reference 3mf not on host"); return; }`.
- **Build commands** (every Run-tests step uses these; the build directory is `build/` and may have been seeded by prior dep builds):
  ```powershell
  $env:Path = "C:\Program Files\CMake\bin;C:\Strawberry\perl\bin;C:\Strawberry\c\bin;$env:Path"
  cmake --build build --target orca-cli  --config Release --parallel 2
  cmake --build build --target cli_tests --config Release --parallel 2
  .\build\tests\cli\Release\cli_tests.exe --order rand --warn NoAssertions
  ```
  `--parallel 2` is deliberate; higher OOMs MSVC PCH on this machine.
- **Commit messages:** `feat(cli):` / `fix(cli):` / `test(cli):` / `docs(cli):` prefixes. Every commit ends with `Co-Authored-By: Claude Opus 4.7 (1M context) <noreply@anthropic.com>` (see existing repo history).
- **--output flag** (spec § 4 preamble): every mutating command accepts `[--output <out>]`. Omitted → in-place. Supplied → side-car write; original untouched. Read-only commands reject `--output` with exit 1.
- **Per-phase smoke gate:** After every phase's commits land green and the smoke recipe section is added to `docs/cli/manual-test.md`, the user runs the recipe, opens the produced 3mf in OrcaSlicer, and reports pass/fail. The plan does **not** continue past a failed gate.

```cpp
#include "output.hpp"
#include <cstdio>
#include <string>

namespace orca_cli {

const char* code_name(ExitCode c)
{
    switch (c) {
        case ExitCode::ok:                  return "ok";
        case ExitCode::usage_error:         return "usage_error";
        case ExitCode::file_not_found:      return "file_not_found";
        case ExitCode::parse_failure:       return "parse_failure";
        case ExitCode::bad_config:          return "bad_config";
        case ExitCode::duplicate_name:      return "duplicate_name";
        case ExitCode::unknown_reference:   return "unknown_reference";
        case ExitCode::invalid_state:       return "invalid_state";
        case ExitCode::invariant_violation: return "invariant_violation";
        case ExitCode::placement_failure:   return "placement_failure";
    }
    return "unknown";
}

static std::string escape_json(std::string_view s)
{
    std::string out;
    out.reserve(s.size() + 2);
    for (char ch : s) {
        switch (ch) {
            case '"':  out += "\\\""; break;
            case '\\': out += "\\\\"; break;
            case '\n': out += "\\n";  break;
            case '\r': out += "\\r";  break;
            case '\t': out += "\\t";  break;
            default:
                if (static_cast<unsigned char>(ch) < 0x20) {
                    char buf[8];
                    std::snprintf(buf, sizeof(buf), "\\u%04x", static_cast<unsigned char>(ch));
                    out += buf;
                } else {
                    out += ch;
                }
        }
    }
    return out;
}

void print_ok(const GlobalOpts& opts, std::string_view message, std::string_view data_json)
{
    if (opts.json) {
        std::string body = "{\"status\":\"ok\",\"code\":\"ok\",\"message\":\"" + escape_json(message) + "\"";
        if (!data_json.empty())
            body += ",\"data\":{" + std::string(data_json) + "}";
        body += "}\n";
        std::fputs(body.c_str(), stdout);
    } else {
        std::fputs("ok: ", stdout);
        std::fwrite(message.data(), 1, message.size(), stdout);
        std::fputc('\n', stdout);
    }
    std::fflush(stdout);
}

void print_err(const GlobalOpts& opts, ExitCode code, std::string_view message)
{
    if (opts.json) {
        std::string body = std::string("{\"status\":\"err\",\"code\":\"") + code_name(code)
                         + "\",\"message\":\"" + escape_json(message) + "\"}\n";
        std::fputs(body.c_str(), stdout);
    } else {
        std::fputs("err: ", stderr);
        std::fputs(code_name(code), stderr);
        std::fputs(": ", stderr);
        std::fwrite(message.data(), 1, message.size(), stderr);
        std::fputc('\n', stderr);
    }
    std::fflush(stdout);
    std::fflush(stderr);
}

} // namespace orca_cli
```

```cpp
#include "cli11/CLI11.hpp"
#include "globals.hpp"
#include "output.hpp"
#include <cstdlib>

namespace { constexpr const char* kVersion = "orca-cli 0.1.0-dev"; }

int main(int argc, char** argv)
{
    CLI::App app{"orca-cli — 3MF composer for OrcaSlicer"};
    app.set_version_flag("--version", kVersion);
    app.require_subcommand(0, 1);   // 0 OK for --version/--help; 1 once subcommands land

    orca_cli::GlobalOpts opts;
    orca_cli::register_global_flags(app, opts);

    try {
        app.parse(argc, argv);
    } catch (const CLI::ParseError& e) {
        return app.exit(e);
    }

    if (app.get_subcommands().empty()) {
        // bare orca-cli (no subcommand) prints help to stderr and returns usage_error
        std::fputs(app.help().c_str(), stderr);
        return static_cast<int>(orca_cli::ExitCode::usage_error);
    }

    // Subcommand dispatch lands in later phases.
    return static_cast<int>(orca_cli::ExitCode::ok);
}
```

```cpp
#include <boost/process.hpp>
#include <boost/process/io.hpp>
#include <chrono>
#include <random>

namespace bp = boost::process;
namespace orca_cli_test {

fs::path ref_3mf() {
    fs::path p(ORCA_CLI_REF_3MF);
    return fs::exists(p) ? p : fs::path{};
}

fs::path stl_dir() { return fs::path(ORCA_CLI_STL_DIR); }
fs::path cli_exe() { return fs::path(ORCA_CLI_EXE); }

fs::path make_temp_dir() {
    using namespace std::chrono;
    auto ts = duration_cast<milliseconds>(steady_clock::now().time_since_epoch()).count();
    std::mt19937_64 rng{std::random_device{}()};
    fs::path p = fs::temp_directory_path()
               / ("orca-cli-test-" + std::to_string(ts) + "-" + std::to_string(rng()));
    fs::create_directories(p);
    return p;
}

fs::path copy_ref_to_temp(const fs::path& temp_dir, const std::string& tag) {
    fs::path dst = temp_dir / ("orca-cli-work-" + tag + ".3mf");
    fs::copy_file(ref_3mf(), dst, fs::copy_options::overwrite_existing);
    return dst;
}
```

```cpp
std::vector<ZipEntry> unzip_to_memory(const std::string& zip_path)
{
    using namespace Slic3r;
    mz_zip_archive archive{};
    if (!open_zip_reader(&archive, zip_path))
        throw InvariantViolation("cannot open archive: " + zip_path);

    std::vector<ZipEntry> out;
    mz_uint n = mz_zip_reader_get_num_files(&archive);
    for (mz_uint i = 0; i < n; ++i) {
        mz_zip_archive_file_stat st;
        if (!mz_zip_reader_file_stat(&archive, i, &st)) continue;
        if (st.m_is_directory) continue;
        std::vector<char> bytes(st.m_uncomp_size);
        if (!mz_zip_reader_extract_to_mem(&archive, i, bytes.data(), bytes.size(), 0))
            continue;
        out.push_back(ZipEntry{ st.m_filename, std::move(bytes) });
    }
    close_zip_reader(&archive);
    return out;
}
```

```cpp
#include <regex>
#include <unordered_set>

void verify_relationships(const std::vector<ZipEntry>& entries)
{
    std::unordered_set<std::string> names;
    for (auto& e : entries) names.insert(e.name);

    std::regex target_re("Target=\"/?([^\"]+)\"");
    for (auto& e : entries) {
        if (e.name.size() < 5 || e.name.substr(e.name.size()-5) != ".rels") continue;
        std::string xml(e.bytes.begin(), e.bytes.end());
        auto begin = std::sregex_iterator(xml.begin(), xml.end(), target_re);
        auto end   = std::sregex_iterator{};
        for (auto it = begin; it != end; ++it) {
            std::string target = (*it)[1].str();
            if (!names.count(target))
                throw InvariantViolation("dangling relationship target: " + target + " (declared in " + e.name + ")");
        }
    }
}
```

```cpp
#include "../cli11/CLI11.hpp"
#include "../output.hpp"
#include "../io.hpp"

#include <boost/filesystem.hpp>

namespace orca_cli::commands {
namespace fs = boost::filesystem;

static int do_project_init(const GlobalOpts& g, const std::string& out, const std::string& tmpl)
{
    if (!fs::exists(tmpl)) {
        print_err(g, ExitCode::file_not_found, "template not found: " + tmpl);
        return int(ExitCode::file_not_found);
    }
    try {
        fs::path tmp = fs::path(out).string() + ".tmp";
        fs::copy_file(tmpl, tmp, fs::copy_options::overwrite_existing);

        // Load + save through the pipeline so the invariant guard runs on the clone.
        auto s = load_project(tmp.string());
        save_project(s, out);   // also removes tmp on its own .tmp path
        fs::remove(tmp);        // belt-and-braces if no rename happened
    } catch (const InvariantViolation& e) {
        print_err(g, ExitCode::invariant_violation, e.what());
        return int(ExitCode::invariant_violation);
    } catch (const std::exception& e) {
        print_err(g, ExitCode::parse_failure, e.what());
        return int(ExitCode::parse_failure);
    }
    print_ok(g, "initialized project at " + out);
    return int(ExitCode::ok);
}

void register_project_subcmd(CLI::App& app, GlobalOpts& g)
{
    auto* project = app.add_subcommand("project", "project-level operations");
    auto* init    = project->add_subcommand("init", "clone a reference 3mf into a new project");

    static std::string out, tmpl;
    init->add_option("out", out, "destination .3mf path")->required();
    init->add_option("--template", tmpl, "reference .3mf to clone")->required();

    init->callback([&g]() { std::exit(do_project_init(g, out, tmpl)); });
}

} // namespace orca_cli::commands
```

```cpp
// In add_object(...) — after STL load + source attribution, before placement.
// Apply object-level transforms (these affect the ModelObject, not per-instance):
if (p.rotate.has_value()) {
    obj->rotate(p.rotate->z(), Vec3d::UnitZ());  // for ax,ay,az: apply per-axis; engineer: see ModelObject::rotate
    // For full ax/ay/az, compose three rotations or use set_rotation(*p.rotate).
}
if (p.scale.has_value()) {
    obj->scale(*p.scale);   // ModelObject::scale takes Vec3d
}

const bool explicit_xform = p.translate.has_value() || p.rotate.has_value() || p.scale.has_value();
const int  n = std::max(1, p.count);

if (explicit_xform) {
    // Per spec § 4.3: "--count N stacks N copies of the loaded STL at the
    // same post-transform position". All N instances share one offset.
    Vec3d offset = p.translate.value_or(Vec3d::Zero());
    for (int i = 0; i < n; ++i) {
        ModelInstance* inst = obj->add_instance();
        inst->set_offset(offset);
        plate->objects_and_instances.emplace_back(
            int(s.model->objects.size() - 1),
            int(obj->instances.size() - 1));
    }
} else {
    // No explicit transform: deterministic per-plate grid via place_in_plate.
    int base_idx = int(plate->objects_and_instances.size());
    for (int i = 0; i < n; ++i) {
        ModelInstance* inst = obj->add_instance();
        Vec3d pos = place_in_plate(bed, base_idx + i, obj->bounding_box().size());
        inst->set_offset(pos);
        plate->objects_and_instances.emplace_back(
            int(s.model->objects.size() - 1),
            int(obj->instances.size() - 1));
    }
}
```

This is the **critical correction over Task 3.2**: as written there, `--count N --translate x,y` would have spread N copies on the grid instead of stacking them at `(x,y)`. The branch above is the inverse — explicit transforms always stack, grid only fires when no transform is supplied.
